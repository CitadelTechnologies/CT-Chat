# CT-Chat

### Install

In your GOPATH src directory :

```shell
	git clone git@github.com:CitadelTechnologies/CT-Chat ct-chat
	cd ct-chat
	go build
```

Then you can get the ct-chat binary

### Usage

In the directory containing the binary :

```shell
	./ct-chat
```
